2 #permalink Buttons

2.1 #permalink Buttons

The following button styles should be used depending on the type of action that is performed. Action - This button should be used for actions which are important, but not necessarily the most important action in the current view Primary action - Used for the most important action in a given context Secondary action - Used for actions that should complement primary actions or visually less important actions

Examples
Default styling
.cfnext-btn-action
Action
.cfnext-btn-primary-action
Primary action
.cfnext-btn-secondary-action
Secondary action
.cfnext-btn-caution
Caution
.cfnext-btn-plain
Plain button 
<button class="[modifier class]">Action</button>
<button class="[modifier class] is-loading">Loading</button>
<button class="[modifier class]" disabled="disabled">Disabled</button>

2.2 #permalink Block button

Use x--block to make the button a block element

Example 

<p>
  <button class="cfnext-btn-action x--block">Block Action Button</button>
</p>
<p>
  <button class="cfnext-btn-action x--block is-loading">Loading</button>
</p>
